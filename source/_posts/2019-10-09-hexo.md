###### 安装 
我参考的这篇[安装教程](https://godweiyang.com/2018/04/13/hexo-blog/)

###### 命令
``` bash
// 生成新页面/文章
hexo new 'newFile'  // source/_post 文件夹下创建一个文件，命名为：newFile.md
hexo new page 'newFile'  // source 文件夹下创建一个名为newFile的文件夹，文件夹下的index.md为页面内容

// 构建静态文件
hexo g

// 启动服务
hexo s
```
